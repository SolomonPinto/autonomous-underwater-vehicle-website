Project Overview

Cutting-edge underwater inspection technology

Project Mission

Our Autonomous Underwater Vehicle represents a breakthrough in maritime inspection technology. This innovative system combines advanced computer vision algorithms with autonomous navigation to detect structural defects and environmental changes in underwater environments.

Key Objectives

  • Develop autonomous navigation system for underwater environments
  • Implement computer vision for rusting detection
  • Create biofouling analysis algorithms
  • Enable real-time data transmission via Telegram
  • Design cost-effective underwater inspection solution
